# NFT Skulls 2022

## Commands

- Compile: npx hardhat compile
- Deploy: npx hardhat --network rinkeby run scripts/deploy.js
- Clean: npx hardhat clean
- Validate contract on etherscan: npx hardhat verify --network rinkeby 0xcontractAddress
